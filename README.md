## The tiny search engine project
* [x] Crawler
* [x] Indexer
* [x] Query engine

### TO BUILD
Type "make". This will recursively build all components under `bin` folder.

### TO RUN
bin/crawler -> bin/indexer -> bin/engine

Alternatively, use `./build_test_run.sh` to automate the whole process.
